Class: Canvas2dRenderer

soundworks/client~Canvas2dRenderer

Base class to extend in order to be used in conjonction with a CanvasView. These classes provide altogether a clean way to manage the update and render cycles of an animation.

Constructor

new Canvas2dRenderer(updatePeriodopt)

Parameters:
Name Type Attributes Default Description
updatePeriod Number <optional>
 0

Logical time (in second) between each subsequent updates. If 0, the update period is slaved on the requestAnimationFrame period (which is appriopriate for most of the use-cases).
