Namespace: server

soundworks/server.server

Server side entry point for a soundworks application.

This object hosts configuration informations, as well as methods to initialize and start the application. It is also responsible for creating the static file (http) server as well as the socket server.

Source:
  • server/core/server.js, line 106

Example

import * as soundworks from 'soundworks/server';
import MyExperience from './MyExperience';

soundworks.server.init(config);
const myExperience = new MyExperience();
soundworks.server.start();
