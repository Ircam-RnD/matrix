sync

Set/Get synchronized mode status. in non sync. mode, the stream audio will start whenever the first audio buffer is downloaded. in sync. mode, the stream audio will start (again whan the audio buffer is downloaded) with an offset in the buffer, as if it started playing exactly when the start() command was issued.
