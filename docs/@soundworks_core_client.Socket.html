Socket

Socket

Simple wrapper with simple pubsub system built on top of ws socket. The abstraction actually open two different sockets:

  • one configured for string (JSON compatible) messages
  • one configured with binaryType=arraybuffer for streaming data more efficiently. The sockets also re-emits all "native" ws events.
