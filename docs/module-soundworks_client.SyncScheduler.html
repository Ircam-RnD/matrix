Class: SyncScheduler

soundworks/client.SyncScheduler

Interface for the client 'sync-scheduler' service.

The sync-scheduler provides a scheduler synchronised among all client using the sync service.

While this service has no direct server counterpart, its dependency on the sync service which requires the existence of a server.
