Class: RawSocket

soundworks/server~RawSocket

Interface for the raw-socket service.

This service creates an additionnal native socket with its binary type set to arraybuffer and focused on performances. It allows the transfert of TypedArray data wrapped with a minimal channel mechanism (up to 256 channels).

The user-defined protocol must follow the convention:

Constructor

new RawSocket()

Source:
  • server/services/RawSocket.js, line 52
Example
const protocol = [
  { channel: 'my-channel', type: 'Float32' }
  // ...
]

Where the `channel` can be any string and the `type` can be interpolated
to any `TypedArray` by concatenating `'Array'` at its end.

__*The service must be used with its client-side counterpart*__
