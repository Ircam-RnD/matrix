    Performance

    Extends:

    src/server/Module~Module → Performance

    [server] Base class used to build a performance on the client side.

    Along with the classic Performance#connect and Performance#disconnect methods, the base class has two additional methods:

    • Performance#enter: called when the client enters the performance (i.e. when the src/client/Performance.js~Performance on the client side calls its src/client/Performance.js~Performance#start method);
    • Performance#exit: called when the client leaves the performance (i.e. when the src/client/Performance.js~Performance on the client side calls its src/client/Performance.js~Performance#done method, or if the client disconnected from the server).

    The base class also keeps track of the clients who are currently in the performance (i.e. who entered but not exited yet) in the array this.clients.

    (See also src/client/Performance.js~Performance on the client side.)
