    Space

    Extends:

    src/client/Module~Module → Space

    [client] Render a Setup data graphically.

    The module never has a view (it displays the graphical representation in a div passed in as an argument of the Space#display method).

    The module finishes its initialization immediately.

    Example:

    const setup = new Setup();
const space = new Space();
const container = document.getElementById('#spaceContainer');

space.display(setup, container);
